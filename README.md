# Perceptron-Classifier-from-Scratch
A 2D & 3D Perceptron Classifier using Python3, NumPy and Matplotlib libraries.

The project consists of two python files.

<br>
1. 2D Percetron Classifier. <br>
perceptron2D.py 
dataset2D.txt
<br>  
Run Code:
  python3 perceptron2D.py
